# github-tools

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Tools

- issue-comment-bot: A bot that comments on issues and PRs, create a new comment or edit an existing one.
